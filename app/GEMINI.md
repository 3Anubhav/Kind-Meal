# Gemini App Module Configuration

This file provides specific instructions for the `app` module.

## Module Overview

This is the main application module for the Registration Form app.

## Key Files

*   `src/main/java/`: Contains the Kotlin source code for the application.
*   `src/main/res/`: Contains the XML layouts, drawables, and other resources.
*   `src/main/AndroidManifest.xml`: The application manifest file.
*   `build.gradle.kts`: The Gradle build script for this module.

## Conventions

*   Use Kotlin for all new code.
*   Follow Android best practices for UI and architecture.
*   Use View Binding or Data Binding for accessing views.
*   Keep UI and business logic separate.
