# Kind-Meal
An Android Application designed to address the growing problem of food wastage by creating a digital bridge between restaurants with surplus food and NGOs or Trusts serving underprivileged communities.
Every day, restaurants, hotels, and food outlets generate large quantities of edible food that often goes to waste. At the same time, NGOs and charitable organizations face difficulties in meeting the nutritional needs of the communities they support. 
Kind Meal provides a unified solution to connect these two sectors efficiently.
The application has been developed in Android Studio using Kotlin and XML, along with Jetpack Compose for modern UI components. 
A custom library mix has been used to improve UI/UX flexibility. 
For the backend, Firebase enables user authentication, real-time database management, and cloud notifications, ensuring smooth and secure operations between donors and NGOs.
In the future, Kind Meal aims to expand beyond restaurants and hotels to include households, events, supermarkets, and catering services, thereby addressing food wastage across the entire food sector.
